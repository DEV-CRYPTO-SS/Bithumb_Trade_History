# Bithumb Trade History Exporter & Analyzer

빗썸(Bithumb) 거래 내역을 한눈에 분석하고 CSV로 내보낼 수 있는 브라우저 북마클렛(Bookmarklet)입니다. 

복잡한 거래 데이터를 시각화하고, 특정 기간의 정산 금액과 수수료를 자동으로 계산해줍니다.

## 주요 기능

- **자동 데이터 로딩**: "더보기" 버튼을 일일이 누를 필요 없이, 목표 날짜까지 자동으로 데이터를 불러옵니다.
- **실시간 필터링**: 거래 구분(매수/매도/입출금 등), 자산별, 특정 검색어 및 기간별로 데이터를 즉시 필터링합니다.
- **상세 통계 분석**: 필터링된 결과에 따른 총 정산금액, 수수료, 매수/매도 횟수 등을 실시간으로 계산하여 대시보드 형태로 보여줍니다.
- **CSV 내보내기**: 분석된 모든 데이터를 엑셀 등에서 활용 가능한 CSV 파일로 원클릭 저장합니다.
- **통계 복사**: 요약된 통계 정보를 클립보드에 복사하여 보고서나 메모에 바로 붙여넣을 수 있습니다.

## 사용 방법

### 방법 1: 북마크렛으로 등록 (권장)
1. 브라우저의 북마크 바에서 우클릭 후 **페이지 추가** 또는 **북마크 추가**를 선택합니다.
2. 이름은 `빗썸 거래내역` 등으로 설정합니다.
3. URL(또는 주소) 칸에 `bithumb_trade_export_bookmarklet_v1.1.js` 파일의 내용을 전체 복사하여 붙여넣습니다.
4. 빗썸 거래내역 페이지에서 해당 북마크를 클릭하면 실행됩니다.

### 방법 2: 개발자 도구 콘솔 사용
1. 빗썸 [거래내역 페이지](https://www.bithumb.com/react/history/trade)에 접속합니다.
2. `F12` 키를 눌러 개발자 도구를 엽니다.
3. `Console` 탭을 선택합니다.
4. `bithumb_trade_export.js`의 전체 코드를 복사하여 붙여넣고 `Enter`를 누릅니다.

## UI 프리뷰

- **대시보드**: 상단에 총 건수, 정산금액, 수수료 등을 그라디언트 카드로 표시
- **필터 섹션**: 검색창, 거래구분/자산 드롭다운, 기간 설정 및 자동 로딩 설정
- **거래 내역 테이블**: 하단에 상세 내역을 테이블 형태로 제공

## 주의 사항

- 이 스크립트는 빗썸 공식 API를 사용하는 것이 아니라 브라우저에 표시된 DOM 요소를 분석하여 동작합니다. 빗썸의 웹사이트 구조가 변경될 경우 작동하지 않을 수 있습니다.
- 개인 거래 데이터는 브라우저 내부에서만 처리되며, 외부 서버로 전송되지 않습니다.
- 대량의 데이터를 불러올 때는 네트워크 환경에 따라 "더보기" 클릭 싸이클이 오랫동안 지속될 수 있습니다.
